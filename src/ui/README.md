# UI Module

This module will contain Streamlit UI components and utilities.

## Planned Functionality

- **Image Upload**: Allow users to upload images for jaundice detection
- **Prediction Display**: Show model predictions with confidence scores
- **Visualization**: Display input images and any relevant visualizations
- **Results Formatting**: Present results in a user-friendly format

## Implementation Plan

1. Create Streamlit interface for image upload
2. Integrate inference module for real-time predictions
3. Design UI layout with clear sections for input and output
4. Add visual feedback and loading indicators
5. Style the interface for a clean, professional look
