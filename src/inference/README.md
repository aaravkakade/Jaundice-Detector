# Inference Module

This module will contain model inference and prediction utilities.

## Planned Functionality

- **Model Loading**: Load trained model checkpoints
- **Image Preprocessing**: Prepare input images for model inference
- **Prediction**: Run inference on single images or batches
- **Post-processing**: Convert model outputs to human-readable predictions
- **Confidence Scores**: Return prediction probabilities/confidence

## Implementation Plan

1. Implement model loading from checkpoint files
2. Create image preprocessing pipeline matching training preprocessing
3. Implement inference function for single images
4. Add batch inference support
5. Format outputs with class labels and confidence scores
