# Streamlit Application

This directory contains the main Streamlit application for the jaundice detection demo.

## Files

- **`app.py`**: Main Streamlit application entry point
  - Will import and use modules from `src/` directory
  - Handles UI layout, user interactions, and displays predictions

## Running the App

Once implemented, run the app with:
```bash
streamlit run app/app.py
```
